```Python
class Solution:
    def numJewelsInStones(self, J, S):
        """
        :type J: str
        :type S: str
        :rtype: int
        """
        cnt = 0
        for stone in S:
            if stone in J:
                cnt += 1
        return cnt
```

题意：字符串J和S，J表示属于宝石的石头，S表示你拥有的石头。返回你拥有的宝石数。
