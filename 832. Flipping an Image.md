```CPP
class Solution {
public:
    vector<vector<int>> flipAndInvertImage(vector<vector<int>>& A) {
        int n = A.size();
        vector<vector<int>> res(n, vector<int>(n, 0));
        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < n; ++j) {
                res[i][j] = 1 - A[i][n-1-j];
            }
        }
        return res;
    }
};
```

题意：1、这道题目给定一个二维的vector，里面的元素是1或者0，要求把每一行的元素第一个和最后一个交换位置，第二个和倒数第二个元素交换位置，依此类推……并且对每个元素都做非操作——把1变成0，把0变成1.
