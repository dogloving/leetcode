```CPP
class Solution {
public:
    int balancedStringSplit(string s) {
        int res = 0, l = 0;
        for (char ch: s) {
            if (ch == 'L') ++l;
            else --l;
            if (l == 0) ++res;
        }
        return res;
    }
};
```

题意：Balaced字符串即有相同数量L和R的字符串。给一个balanced字符串s，将其分成尽量多的子balanced字符串。
