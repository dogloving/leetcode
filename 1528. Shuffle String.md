```CPP
// time: O(n), space: O(n)
class Solution {
public:
    string restoreString(string s, vector<int>& indices) {
        string res(s.size(),'0');
        for (int i = 0; i < indices.size(); ++i) res[indices[i]] = s[i];
        return res;
    }
};
```

题意：给一个字符串s和整数数组indices，原本s中第i个字符移动到第indices[i]的位置。求移动后的字符串。

思路：简单。
