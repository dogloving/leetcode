```CPP
// time: O(logn), space: O(1)
class Solution {
public:
    bool isPowerOfFour(int num) {
        if (num==0) return false;
        if (num==1) return true;
        while (num!=1) {
            if (num%4) return false;
            num /= 4;
        }
        return true;
    }
};
```

题意：检查一个数是否为4的幂。

思路：简单。
