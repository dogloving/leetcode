```CPP
class Solution {
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> res;
        int n = nums.size();
        for (int i = 0; i < pow(2, n); ++i) {
            vector<int> tmp;
            int idx = 0;
            int ii = i;
            while (idx<n) {
                if (ii%2) tmp.push_back(nums[idx]);
                ii /= 2;
                ++idx;
            }
            res.push_back(tmp);
        }
        return res;
    }
};
```

题意：给出所有数都不同的数组，求所有可能的数组合。

思路：假设有n个数，那么共2^n个set。我们从0加到2^n-1，将对应位上为1的数加入。
