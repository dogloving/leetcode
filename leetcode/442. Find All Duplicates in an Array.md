```cpp
class Solution {
public:
    void swap(int& a,int& b) {
        int c = a;
        a = b;
        b = c;
    }
    vector<int> findDuplicates(vector<int>& nums) {
        vector<int> res;
        int i = 0;
        while(i < nums.size()) {
            if(nums[i] != nums[nums[i]-1]) swap(nums[i],nums[nums[i]-1]);
            else ++i;
        }
        for(int i = 0; i < nums.size(); ++i) {
            if(nums[i] != i+1) res.push_back(nums[i]);
        }
        return res;
    }
};
```
题意：给一个整数数组a，1<=a[i]<=n，有些元素出现2次有些只出现1次。找到所有出现两次的数。要求O(n)时间复杂度，O(1)空间复杂度。

思路：如果能够使用额外空间的话会比较简单。但是如果不适用额外空间的话思考如何将每个值出现次数记录下来成了一个问题。我们发现其实如果对这些数字进行排序，本质是某些数字被另一些数字替换掉了，而替换者就是我们要找的重复的数。所以问题就转换成了将这些数放到原本属于他们的位置上去。我们对数组进行遍历，假设到了第i个数，这个数的值是nums[i],nums[i]原本应该在nums[i]-1的位置上，所以我们把这两个位置上的数交换。如果这两个位置上的数原本就相等，说明第nums[i]-1的位置上已经是正确的数了。交换之后，我们重新对第i个位置上的数进行操作。
