```CPP
class Solution {
public:
    int subtractProductAndSum(int n) {
        int p = 1, s = 0;
        while(n) {
            int x = n % 10;
            p *= x;
            s += x;
            n /= 10;
        }
        return p-s;
    }
};
```

题意：n个人分成多组，每个人有自己的ID0~n-1。整数数组groupSizes，groupSizes[i]表示第i个人所在的group有多少人。返回各个组有哪些人。可能有多种分组结果，返回一种即可。
